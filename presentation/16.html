Scalability Architecture

Core Principle: Modular Scaling

Designed with isolated domains (Detection, Simulation, Routing) and API-first contracts—allowing us to upgrade any service independently without side effects.

Hybrid Architecture

Live Mode (Direct)

Ultra-low latency inference for demo feeds.

Scalable Mode (Queues)

Infinite scaling for city-wide camera streams.

Scaling Analogy

API = The Cashier Takes orders instantly. Doesn't cook.
AI Workers = The Cooks Processes YOLO inference in the back.
“Because the cashier doesn't cook, the system never bottlenecks, allowing us to process thousands of cameras simultaneously.”
