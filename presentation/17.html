Roadmap to Production

Distributed Persistence

Move in-memory state to Redis/PostgreSQL for global synchronization across city nodes.

Async Inference Pipeline

Put heavy computer-vision tasks into Celery/Kafka workers for high-volume non-blocking flow.

GPU Microservices

Serve model inference as separate micro-services using dedicated GPU clusters.

Management Layer

Add API Gateways, Rate Limiting, and Prometheus/Grafana for enterprise-grade observability.
